(isometric-tiled-map path unit & options)

Returns an IsometricTiledMapRenderer with the tiled map file at path and unit scale.

(isometric-tiled-map "level1.tmx" (/ 1 8))
Options
:render-tile-layer^TiledMapTileLayer layer

(defn isometric-tiled-map*
  [path unit]
  (IsometricTiledMapRenderer. ^TiledMap (if (string? path)
                                          (tiled-map* path)
                                          path)
                              (float unit)))
(defmacro isometric-tiled-map
  [path unit & options]
  `(u/calls! ^IsometricTiledMapRenderer (isometric-tiled-map* ~path ~unit)
             ~@options))
