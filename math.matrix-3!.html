(matrix-3! object k & options)

Calls a single method on a matrix-3.

Options
:det
:get-rotation
:get-rotation-rad
:get-scale^Vector2 scale
:get-translation^Vector2 position
:get-values
Get the values in this matrix.
:idt
Sets this matrix to the identity matrix
:inv
Inverts this matrix given that the determinant is != 0.
:mul^Matrix3 m
Postmultiplies this matrix with the provided matrix and stores the result in this matrix. For example: 
 A.mul(B) results in A := AB
:mul-left^Matrix3 m
Premultiplies this matrix with the provided matrix and stores the result in this matrix. For example: 
 A.mulLeft(B) results in A := BA
:rotate^float degrees
Postmultiplies this matrix with a (counter-clockwise) rotation matrix. Postmultiplication is also used by OpenGL ES' 1.x glTranslate/glRotate/glScale.
:rotate-rad^float radians
Postmultiplies this matrix with a (counter-clockwise) rotation matrix. Postmultiplication is also used by OpenGL ES' 1.x glTranslate/glRotate/glScale.
:scale^float scale-x, ^float scale-y
Postmultiplies this matrix with a scale matrix. Postmultiplication is also used by OpenGL ES' 1.x glTranslate/glRotate/glScale.
:scale^Vector2 scale
Postmultiplies this matrix with a scale matrix. Postmultiplication is also used by OpenGL ES' 1.x glTranslate/glRotate/glScale.
:scl^float scale
Scale the matrix in the both the x and y components by the scalar value.
:scl^Vector2 scale
Scale this matrix using the x and y components of the vector but leave the rest of the matrix alone.
:scl^Vector3 scale
Scale this matrix using the x and y components of the vector but leave the rest of the matrix alone.
:set^Matrix3 mat
Copies the values from the provided matrix to this matrix.
:set^Affine2 affine
Copies the values from the provided affine matrix to this matrix. The last row is set to (0, 0, 1).
:set^Matrix4 mat
Sets this 3x3 matrix to the top left 3x3 corner of the provided 4x4 matrix.
:set^float[] values
Sets the matrix to the given matrix as a float array. The float array must have at least 9 elements; the first 9 will be copied.
:set-to-rotation^float degrees
Sets this matrix to a rotation matrix that will rotate any vector in counter-clockwise direction around the z-axis.
:set-to-rotation^Vector3 axis, ^float degrees
:set-to-rotation^Vector3 axis, ^float cos, ^float sin
:set-to-rotation-rad^float radians
Sets this matrix to a rotation matrix that will rotate any vector in counter-clockwise direction around the z-axis.
:set-to-scaling^float scale-x, ^float scale-y
Sets this matrix to a scaling matrix.
:set-to-scaling^Vector2 scale
Sets this matrix to a scaling matrix.
:set-to-translation^float x, ^float y
Sets this matrix to a translation matrix.
:set-to-translation^Vector2 translation
Sets this matrix to a translation matrix.
:to-string
:translate^float x, ^float y
Postmultiplies this matrix by a translation matrix. Postmultiplication is also used by OpenGL ES' 1.x glTranslate/glRotate/glScale.
:translate^Vector2 translation
Postmultiplies this matrix by a translation matrix. Postmultiplication is also used by OpenGL ES' 1.x glTranslate/glRotate/glScale.
:transpose
Transposes the current matrix.
:trn^Vector2 vector
Adds a translational component to the matrix in the 3rd column. The other columns are untouched.
:trn^float x, ^float y
Adds a translational component to the matrix in the 3rd column. The other columns are untouched.
:trn^Vector3 vector
Adds a translational component to the matrix in the 3rd column. The other columns are untouched.

(defmacro matrix-3!
  [object k & options]
  `(u/call! ^Matrix3 ~object ~k ~@options))
