(bounding-box! object k & options)

Calls a single method on a bounding-box.

Options
:clr
Sets the minimum and maximum vector to zeros.
:contains^BoundingBox b
Returns whether the given bounding box is contained in this bounding box.
:contains^Vector3 v
Returns whether the given vector is contained in this bounding box.
:ext^Vector3 point
Extends the bounding box to incorporate the given {@link Vector3}.
:ext^BoundingBox a_bounds
Extends this bounding box by the given bounding box.
:ext^BoundingBox bounds, ^Matrix4 transform
Extends this bounding box by the given transformed bounding box.
:ext^float x, ^float y, ^float z
Extends the bounding box by the given vector.
:get-center
:get-center^Vector3 out
:get-center-x
:get-center-y
:get-center-z
:get-corner000^Vector3 out
:get-corner001^Vector3 out
:get-corner010^Vector3 out
:get-corner011^Vector3 out
:get-corner100^Vector3 out
:get-corner101^Vector3 out
:get-corner110^Vector3 out
:get-corner111^Vector3 out
:get-corners
:get-depth
:get-dimensions
:get-dimensions^Vector3 out
:get-height
:get-max
:get-max^Vector3 out
:get-min
:get-min^Vector3 out
:get-width
:inf
Sets the minimum and maximum vector to positive and negative infinity.
:intersects^BoundingBox b
Returns whether the given bounding box is intersecting this bounding box (at least one point in).
:is-valid
Returns whether this bounding box is valid. This means that {@link #max} is greater than {@link #min}.
:mul^Matrix4 transform
Multiplies the bounding box by the given matrix. This is achieved by multiplying the 8 corner points and then calculating the minimum and maximum vectors from the transformed points.
:set^BoundingBox bounds
Sets the given bounding box.
:set^Vector3 minimum, ^Vector3 maximum
Sets the given minimum and maximum vector.
:set^com.badlogic.gdx.math.Vector3[] points
Sets the bounding box minimum and maximum vector from the given points.
:set^java.util.List points
Sets the bounding box minimum and maximum vector from the given points.
:to-string
