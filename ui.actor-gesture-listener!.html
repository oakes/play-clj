(actor-gesture-listener! screen k & options)

Calls a single method on the ActorGestureListener in the screen.

Options
:fling^InputEvent event, ^float velocity-x, ^float velocity-y, ^int button
:get-gesture-detector
:get-touch-down-target
:handle^Event e
:long-press^Actor actor, ^float x, ^float y
If true is returned, additional gestures will not be triggered. No event is provided because this event is triggered by time passing, not by an InputEvent.
:pan^InputEvent event, ^float x, ^float y, ^float delta-x, ^float delta-y
The delta is the difference in stage coordinates since the last pan.
:pinch^InputEvent event, ^Vector2 initial-pointer1, ^Vector2 initial-pointer2, ^Vector2 pointer1, ^Vector2 pointer2
:tap^InputEvent event, ^float x, ^float y, ^int count, ^int button
:touch-down^InputEvent event, ^float x, ^float y, ^int pointer, ^int button
:touch-up^InputEvent event, ^float x, ^float y, ^int pointer, ^int button
:zoom^InputEvent event, ^float initial-distance, ^float distance
Source
(defmacro actor-gesture-listener!
  [screen k & options]
  `(let [listeners# (u/get-obj ~screen :ui-listeners)
         ^ActorGestureListener object#
         (u/get-obj listeners# :actor-gesture-listener)]
     (u/call! object# ~k ~@options)))
