(files! k & options)

Calls a single method on Gdx.files.

(files! :internal "image.png")
Options
:absolute^String path
Convenience method that returns a {@link FileType#Absolute} file handle.
:classpath^String path
Convenience method that returns a {@link FileType#Classpath} file handle.
:external^String path
Convenience method that returns a {@link FileType#External} file handle.
:get-external-storage-path
Returns the external storage path directory. This is the SD card on Android and the home directory of the current user on the desktop.
:get-file-handle^String path, ^Files.FileType type
Returns a handle representing a file or directory.
:get-local-storage-path
Returns the local storage path directory. This is the private files directory on Android and the directory of the jar on the desktop.
:internal^String path
Convenience method that returns a {@link FileType#Internal} file handle.
:is-external-storage-available
Returns true if the external storage is ready for file IO. Eg, on Android, the SD card is not available when mounted for use with a PC.
:is-local-storage-available
Returns true if the local storage is ready for file IO.
:local^String path
Convenience method that returns a {@link FileType#Local} file handle.
Source
(defmacro files!
  [k & options]
  `(u/call! ^Files (Gdx/files) ~k ~@options))
