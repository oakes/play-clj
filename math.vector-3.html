(vector-3 x y z & options)

Returns a Vector3. The values can be retrieved with x, y and z, and modified with x!, y!, and z!.

Options
:add^Vector3 vector
:add^float x, ^float y, ^float z
Adds the given vector to this component
:add^float values
Adds the given value to all three components of the vector.
:clamp^float min, ^float max
:cpy
:crs^Vector3 vector
Sets this vector to the cross product between it and the other vector.
:crs^float x, ^float y, ^float z
Sets this vector to the cross product between it and the other vector.
:dot^Vector3 vector
:dot^float x, ^float y, ^float z
Returns the dot product between this and the given vector.
:dst^Vector3 vector
:dst^float x, ^float y, ^float z
:dst2^Vector3 point
:dst2^float x, ^float y, ^float z
Returns the squared distance between this point and the given point
:epsilon-equals^Vector3 other, ^float epsilon
:epsilon-equals^float x, ^float y, ^float z, ^float epsilon
Compares this vector with the other vector, using the supplied epsilon for fuzzy equality testing.
:equals^Object obj
:from-string^String v
Sets this {@code Vector3} to the value represented by the specified string according to the format of {@link #toString()}.
:has-opposite-direction^Vector3 vector
:has-same-direction^Vector3 vector
:hash-code
:idt^Vector3 vector
:interpolate^Vector3 target, ^float alpha, ^Interpolation interpolator
:is-collinear^Vector3 other, ^float epsilon
:is-collinear^Vector3 other
:is-collinear-opposite^Vector3 other, ^float epsilon
:is-collinear-opposite^Vector3 other
:is-on-line^Vector3 other, ^float epsilon
:is-on-line^Vector3 other
:is-perpendicular^Vector3 vector
:is-perpendicular^Vector3 vector, ^float epsilon

:is-unit^float margin
:is-zero
:is-zero^float margin
:len
:len2
:lerp^Vector3 target, ^float alpha
:limit^float limit
:limit2^float limit2
:mul^Matrix4 matrix
Left-multiplies the vector by the given matrix, assuming the fourth (w) component of the vector is 1.
:mul^Matrix3 matrix
Left-multiplies the vector by the given matrix.
:mul^Quaternion quat
Multiplies the vector by the given {@link Quaternion}.
:mul-add^Vector3 vec, ^float scalar
:mul-add^Vector3 vec, ^Vector3 mul-vec
:mul4x3^float[] matrix
Left-multiplies the vector by the given 4x3 column major matrix. The matrix should be composed by a 3x3 matrix representing rotation and scale plus a 1x3 matrix representing the translation.
:nor
:prj^Matrix4 matrix
Multiplies this vector by the given matrix dividing by w, assuming the fourth (w) component of the vector is 1. This is mostly used to project/unproject vectors via a perspective projection matrix.
:rot^Matrix4 matrix
Multiplies this vector by the first three columns of the matrix, essentially only applying rotation and scaling.
:rotate^float degrees, ^float axis-x, ^float axis-y, ^float axis-z
Rotates this vector by the given angle in degrees around the given axis.
:rotate^Vector3 axis, ^float degrees
Rotates this vector by the given angle in degrees around the given axis.
:rotate-rad^float radians, ^float axis-x, ^float axis-y, ^float axis-z
Rotates this vector by the given angle in radians around the given axis.
:rotate-rad^Vector3 axis, ^float radians
Rotates this vector by the given angle in radians around the given axis.
:scl^float scalar
:scl^Vector3 other
:scl^float vx, ^float vy, ^float vz
Scales this vector by the given values
:set^float x, ^float y, ^float z
Sets the vector to the given components
:set^Vector3 vector
:set^float[] values
Sets the components from the array. The array must have at least 3 elements
:set^Vector2 vector, ^float z
Sets the components of the given vector and z-component
:set-from-spherical^float azimuthal-angle, ^float polar-angle
Sets the components from the given spherical coordinate
:set-length^float len
:set-length2^float len2
:set-to-random-direction
:set-zero
:slerp^Vector3 target, ^float alpha
Spherically interpolates between this vector and the target vector by alpha which is in the range [0,1]. The result is stored in this vector.
:sub^Vector3 a_vec
:sub^float x, ^float y, ^float z
Subtracts the other vector from this vector.
:sub^float value
Subtracts the given value from all components of this vector
:to-string
Converts this {@code Vector3} to a string in the format {@code (x,y,z)}.
:tra-mul^Matrix4 matrix
Multiplies the vector by the transpose of the given matrix, assuming the fourth (w) component of the vector is 1.
:tra-mul^Matrix3 matrix
Multiplies the vector by the transpose of the given matrix.
:unrotate^Matrix4 matrix
Multiplies this vector by the transpose of the first three columns of the matrix. Note: only works for translation and rotation, does not work for scaling. For those, use {@link #rot(Matrix4)} with {@link Matrix4#inv()}.
:untransform^Matrix4 matrix
Translates this vector in the direction opposite to the translation of the matrix and the multiplies this vector by the transpose of the first three columns of the matrix. Note: only works for translation and rotation, does not work for scaling. For those, use {@link #mul(Matrix4)} with {@link Matrix4#inv()}.

(defn vector-3*
  ([]
    (Vector3.))
  ([x y z]
    (Vector3. x y z)))
(defmacro vector-3
  [x y z & options]
  `(u/calls! ^Vector3 (vector-3* ~x ~y ~z) ~@options))
