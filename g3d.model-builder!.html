:end
End building the model.
:manage^Disposable disposable
Add the {@link Disposable} object to the model, causing it to be disposed when the model is disposed.
:node
Add a node to the model. Use any of the part(...) method to add a NodePart.
:node^String id, ^Model model
Adds the nodes of the specified model to a new node of the model being build. After this method the given model can no longer be used. Do not call the {@link Model#dispose()} method on that model.
:part^MeshPart meshpart, ^Material material
Adds the specified MeshPart to the current Node. The Mesh will be managed by the model and disposed when the model is disposed. The resources the Material might contain are not managed, use {@link #manage(Disposable)} to add those to the model.
:part^String id, ^Mesh mesh, ^int primitive-type, ^int offset, ^int size, ^Material material
Adds the specified mesh part to the current node. The Mesh will be managed by the model and disposed when the model is disposed. The resources the Material might contain are not managed, use {@link #manage(Disposable)} to add those to the model.
:part^String id, ^Mesh mesh, ^int primitive-type, ^Material material
Adds the specified mesh part to the current node. The Mesh will be managed by the model and disposed when the model is disposed. The resources the Material might contain are not managed, use {@link #manage(Disposable)} to add those to the model.
:part^String id, ^int primitive-type, ^VertexAttributes attributes, ^Material material
Creates a new MeshPart within the current Node and returns a {@link MeshPartBuilder} which can be used to build the shape of the part. If possible a previously used {@link MeshPartBuilder} will be reused, to reduce the number of mesh binds. Therefore you can only build one part at a time. The resources the Material might contain are not managed, use {@link #manage(Disposable)} to add those to the model.
:part^String id, ^int primitive-type, ^long attributes, ^Material material
Creates a new MeshPart within the current Node and returns a {@link MeshPartBuilder} which can be used to build the shape of the part. If possible a previously used {@link MeshPartBuilder} will be reused, to reduce the number of mesh binds. Therefore you can only build one part at a time. The resources the Material might contain are not managed, use {@link #manage(Disposable)} to add those to the model.
