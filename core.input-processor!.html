(input-processor! screen k & options)

Calls a single method on the InputProcessor in the screen.

Options
:key-down^int keycode
Called when a key was pressed
:key-typed^char character
Called when a key was typed
:key-up^int keycode
Called when a key was released
:mouse-moved^int screen-x, ^int screen-y
Called when the mouse was moved without any buttons being pressed. Will not be called on iOS.
:scrolled^int amount
Called when the mouse wheel was scrolled. Will not be called on iOS.
:touch-down^int screen-x, ^int screen-y, ^int pointer, ^int button
Called when the screen was touched or a mouse button was pressed. The button parameter will be {@link Buttons#LEFT} on iOS.
:touch-dragged^int screen-x, ^int screen-y, ^int pointer
Called when a finger or the mouse was dragged.
:touch-up^int screen-x, ^int screen-y, ^int pointer, ^int button
Called when a finger was lifted or a mouse button was released. The button parameter will be {@link Buttons#LEFT} on iOS.
Source
(defmacro input-processor!
  [screen k & options]
  `(let [listeners# (u/get-obj ~screen :input-listeners)
         ^InputProcessor object# (u/get-obj listeners# :input-processor)]
     (u/call! object# ~k ~@options)))
