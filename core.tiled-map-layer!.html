(tiled-map-layer! object k & options)

Calls a single method on a tiled-map-layer.

(tiled-map-layer! (tiled-map-layer screen "water")
                  :set-cell 0 0 nil)
Options
:get-cell^int x, ^int y
:get-height
:get-tile-height
:get-tile-width
:get-width
:set-cell^int x, ^int y, ^TiledMapTileLayer.Cell cell
Sets the {@link Cell} at the given coordinates.
Source
(defmacro tiled-map-layer!
  [object k & options]
  `(u/call! ^TiledMapTileLayer (cast TiledMapTileLayer ~object) ~k ~@options))
