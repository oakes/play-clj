(body-def k & options)

Returns a BodyDef.

(body-def :dynamic)
Options
:active^boolean value
Does this body start out active?
:allow-sleep^boolean value
Set this flag to false if this body should never fall asleep. Note that this increases CPU usage.
:angle^float value
The world angle of the body in radians.
:angular-damping^float value
Angular damping is use to reduce the angular velocity. The damping parameter can be larger than 1.0f but the damping effect becomes sensitive to the time step when the damping parameter is large.
:angular-velocity^float value
The angular velocity of the body.
:awake^boolean value
Is this body initially awake or sleeping?
:bullet^boolean value
Is this a fast moving body that should be prevented from tunneling through other moving bodies? Note that all bodies are prevented from tunneling through kinematic and static bodies. This setting is only considered on dynamic bodies.
:fixed-rotation^boolean value
Should this body be prevented from rotating? Useful for characters.
:gravity-scale^float value
Scale the gravity applied to this body.
:linear-damping^float value
Linear damping is use to reduce the linear velocity. The damping parameter can be larger than 1.0f but the damping effect becomes sensitive to the time step when the damping parameter is large.
:linear-velocity^Vector2 value
The linear velocity of the body's origin in world co-ordinates.
:position^Vector2 value
The world position of the body. Avoid creating bodies at the origin since this can lead to many overlapping shapes.
:type^BodyDef.BodyType value
The body type: static, kinematic, or dynamic. Note: if a dynamic body would have zero mass, the mass is set to one.
Source
(defmacro body-def
  [k & options]
  `(let [^BodyDef object# (BodyDef.)]
     (u/fields! object# :type ~(body-type k) ~@options)))
