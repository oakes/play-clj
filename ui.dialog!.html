(dialog! entity k & options)

Calls a single method on a dialog.

Options
:button^String text
Adds a text button to the button table. Null will be passed to {@link #result(Object)} if this button is clicked. The dialog must have been constructed with a skin to use this method.
:button^String text, ^Object object
Adds a text button to the button table. The dialog must have been constructed with a skin to use this method.
:button^String text, ^Object object, ^TextButton.TextButtonStyle button-style
Adds a text button to the button table.
:button^Button button
Adds the given button to the button table.
:button^Button button, ^Object object
Adds the given button to the button table.
:cancel
:get-button-table
:get-content-table
:hide^Action action
Hides the dialog with the given action and then removes it from the stage.
:hide
Hides the dialog. Called automatically when a button is clicked. The default implementation fades out the dialog over 400 milliseconds and then removes it from the stage.
:key^int keycode, ^Object object
If this key is pressed, {@link #result(Object)} is called with the specified object.
:set-object^Actor actor, ^Object object
:show^Stage stage, ^Action action
{@link #pack() Packs} the dialog and adds it to the stage with custom action which can be null for instant show
:show^Stage stage
{@link #pack() Packs} the dialog and adds it to the stage, centered with default fadeIn action
:text^String text
Adds a label to the content table. The dialog must have been constructed with a skin to use this method.
:text^String text, ^Label.LabelStyle label-style
Adds a label to the content table.
:text^Label label
Adds the given Label to the content table
Source
(defmacro dialog!
  [entity k & options]
  `(u/call! ^Dialog (u/get-obj ~entity :object) ~k ~@options))
