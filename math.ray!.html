(ray! object k & options)

Calls a single method on a ray.

Options
:cpy
:equals^Object o
:get-end-point^Vector3 out, ^float distance
Returns the endpoint given the distance. This is calculated as startpoint + distance * direction.
:hash-code
:mul^Matrix4 matrix
Multiplies the ray by the given matrix. Use this to transform a ray into another coordinate system.
:set^Vector3 origin, ^Vector3 direction
Sets the starting position and the direction of this ray.
:set^float x, ^float y, ^float z, ^float dx, ^float dy, ^float dz
Sets this ray from the given starting position and direction.
:set^Ray ray
Sets the starting position and direction from the given ray
:to-string
{@inheritDoc}
Source
(defmacro ray!
  [object k & options]
  `(u/call! ^Ray ~object ~k ~@options))
