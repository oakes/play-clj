(scroll-pane! entity k & options)

Calls a single method on a scroll-pane.

Options
:act^float delta
:add-actor^Actor actor
:add-actor-after^Actor actor-after, ^Actor actor
:add-actor-at^int index, ^Actor actor
:add-actor-before^Actor actor-before, ^Actor actor
:cancel
If currently scrolling by tracking a touch down, stop scrolling.
:draw^Batch batch, ^float parent-alpha
:draw-debug^ShapeRenderer shapes
:fling^float fling-time, ^float velocity-x, ^float velocity-y
Generate fling gesture.
:get-max-x
Returns the maximum scroll value in the x direction.
:get-max-y
Returns the maximum scroll value in the y direction.
:get-min-height
:get-min-width
:get-pref-height
:get-pref-width
:get-scroll-bar-height
:get-scroll-bar-width
:get-scroll-height
Returns the height of the scrolled viewport.
:get-scroll-percent-x
:get-scroll-percent-y
:get-scroll-width
Returns the width of the scrolled viewport.
:get-scroll-x
Returns the x scroll position in pixels, where 0 is the left of the scroll pane.
:get-scroll-y
Returns the y scroll position in pixels, where 0 is the top of the scroll pane.
:get-style
Returns the scroll pane's style. Modifying the returned style may not have an effect until {@link #setStyle(ScrollPaneStyle)} is called.
:get-variable-size-knobs
:get-velocity-x
Gets the flick scroll y velocity.
:get-velocity-y
Gets the flick scroll y velocity.
:get-visual-scroll-percent-x
:get-visual-scroll-percent-y
:get-visual-scroll-x
:get-visual-scroll-y
:get-widget
Returns the actor embedded in this scroll pane, or null.
:hit^float x, ^float y, ^boolean touchable
:is-bottom-edge
:is-dragging
:is-flinging
:is-force-scroll-x
:is-force-scroll-y
:is-left-edge
:is-panning
:is-right-edge
:is-scroll-x
Returns true if the widget is larger than the scroll pane horizontally.
:is-scroll-y
Returns true if the widget is larger than the scroll pane vertically.
:is-top-edge
:layout
:remove-actor^Actor actor
:scroll-to^float x, ^float y, ^float width, ^float height
Sets the scroll offset so the specified rectangle is fully in view, if possible. Coordinates are in the scroll pane widget's coordinate system.
:scroll-to^float x, ^float y, ^float width, ^float height, ^boolean center-horizontal, ^boolean center-vertical
Sets the scroll offset so the specified rectangle is fully in view, and optionally centered vertically and/or horizontally, if possible. Coordinates are in the scroll pane widget's coordinate system.
:set-cancel-touch-focus^boolean cancel-touch-focus
When true (default), the {@link Stage#cancelTouchFocus()} touch focus} is cancelled when flick scrolling begins. This causes widgets inside the scrollpane that have received touchDown to receive touchUp when flick scrolling begins.
:set-clamp^boolean clamp
For flick scroll, prevents scrolling out of the widget's bounds. Default is true.
:set-fade-scroll-bars^boolean fade-scroll-bars
When true the scrollbars don't reduce the scrollable size and fade out after some time of not being used.
:set-flick-scroll^boolean flick-scroll
:set-flick-scroll-tap-square-size^float half-tap-square-size
:set-fling-time^float fling-time
For flick scroll, sets the amount of time in seconds that a fling will continue to scroll. Default is 1.
:set-force-scroll^boolean x, ^boolean y
Forces enabling scrollbars (for non-flick scroll) and overscrolling (for flick scroll) in a direction, even if the contents do not exceed the bounds in that direction.
:set-overscroll^boolean overscroll-x, ^boolean overscroll-y
For flick scroll, if true the widget can be scrolled slightly past its bounds and will animate back to its bounds when scrolling is stopped. Default is true.
:set-scroll-bar-positions^boolean bottom, ^boolean right
Set the position of the vertical and horizontal scroll bars.
:set-scroll-percent-x^float percent-x
:set-scroll-percent-y^float percent-y
:set-scroll-x^float pixels
:set-scroll-y^float pixels
:set-scrollbars-on-top^boolean scrollbars-on-top
When false (the default), the widget is clipped so it is not drawn under the scrollbars. When true, the widget is clipped to the entire scroll pane bounds and the scrollbars are drawn on top of the widget. If {@link #setFadeScrollBars(boolean)} is true, the scroll bars are always drawn on top.
:set-scrolling-disabled^boolean x, ^boolean y
Disables scrolling in a direction. The widget will be sized to the FlickScrollPane in the disabled direction.
:set-smooth-scrolling^boolean smooth-scrolling
:set-style^ScrollPane.ScrollPaneStyle style
:set-variable-size-knobs^boolean variable-size-knobs
If true, the scroll knobs are sized based on {@link #getMaxX()} or {@link #getMaxY()}. If false, the scroll knobs are sized based on {@link Drawable#getMinWidth()} or {@link Drawable#getMinHeight()}. Default is true.
:set-velocity-x^float velocity-x
:set-velocity-y^float velocity-y
:set-widget^Actor widget
Sets the {@link Actor} embedded in this scroll pane.
:setup-fade-scroll-bars^float fade-alpha-seconds, ^float fade-delay-seconds
:setup-overscroll^float distance, ^float speed-min, ^float speed-max
For flick scroll, sets the overscroll distance in pixels and the speed it returns to the widget's bounds in seconds. Default is 50, 30, 200.
:to-string
:update-visual-scroll
Sets the visual scroll amount equal to the scroll amount. This can be used when setting the scroll amount without animating.
Source
(defmacro scroll-pane!
  [entity k & options]
  `(u/call! ^ScrollPane (u/get-obj ~entity :object) ~k ~@options))
