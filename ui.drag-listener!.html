(drag-listener! screen k & options)

Calls a single method on the DragListener in the screen.

Options
:cancel
:drag^InputEvent event, ^float x, ^float y, ^int pointer
:drag-start^InputEvent event, ^float x, ^float y, ^int pointer
:drag-stop^InputEvent event, ^float x, ^float y, ^int pointer
:get-button
:get-delta-x
Returns the amount on the x axis that the touch has been dragged since the last drag event.
:get-delta-y
Returns the amount on the y axis that the touch has been dragged since the last drag event.
:get-stage-touch-down-x
:get-stage-touch-down-y
:get-tap-square-size
:get-touch-down-x
:get-touch-down-y
:is-dragging
Returns true if a touch has been dragged outside the tap square.
:set-button^int button
Sets the button to listen for, all other buttons are ignored. Default is {@link Buttons#LEFT}. Use -1 for any button.
:set-tap-square-size^float half-tap-square-size
:touch-down^InputEvent event, ^float x, ^float y, ^int pointer, ^int button
:touch-dragged^InputEvent event, ^float x, ^float y, ^int pointer
:touch-up^InputEvent event, ^float x, ^float y, ^int pointer, ^int button
Source
(defmacro drag-listener!
  [screen k & options]
  `(let [listeners# (u/get-obj ~screen :ui-listeners)
         ^DragListener object# (u/get-obj listeners# :drag-listener)]
     (u/call! object# ~k ~@options)))
