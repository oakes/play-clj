(map-object! object k & options)

Calls a single method on a map-object.

(map-object! (map-object :rectangle) :get-rectangle)
Options
:get-color
:get-name
:get-opacity
:get-properties
:is-visible
:set-color^Color color
:set-name^String name
:set-opacity^float opacity
:set-visible^boolean visible
Source
(defmacro map-object!
  [object k & options]
  `(u/call! ^MapObject ~object ~k ~@options))
