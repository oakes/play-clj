(tiled-map-cell layer x y & options)

Returns a TiledMapTileLayer.Cell from the layer at position x and y.

(tiled-map-cell (tiled-map-layer screen "water") 0 0)
Options
:get-flip-horizontally
:get-flip-vertically
:get-rotation
:get-tile
:set-flip-horizontally^boolean flip-horizontally
Sets whether to flip the tile horizontally.
:set-flip-vertically^boolean flip-vertically
Sets whether to flip the tile vertically.
:set-rotation^int rotation
Sets the rotation of this cell, in degrees.
:set-tile^TiledMapTile tile
Sets the tile to be used for this cell.

(defn tiled-map-cell*
  ([]
    (TiledMapTileLayer$Cell.))
  ([^TiledMapTileLayer layer x y]
    (.getCell layer x y)))
(defmacro tiled-map-cell
  [layer x y & options]
  `(let [^TiledMapTileLayer$Cell object# (tiled-map-cell* ~layer ~x ~y)]
     (u/calls! object# ~@options)))
